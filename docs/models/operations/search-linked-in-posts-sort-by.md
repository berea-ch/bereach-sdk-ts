# SearchLinkedInPostsSortBy

Sort order. 'relevance' (default) returns the most relevant posts. 'date' returns the most recent posts first.

## Example Usage

```typescript
import { SearchLinkedInPostsSortBy } from "bereach/models/operations";

let value: SearchLinkedInPostsSortBy = "relevance";
```

## Values

```typescript
"relevance" | "date"
```
