# Actions

## Overview

Endpoints for performing LinkedIn actions (visit, connect, message, like, reply, publish)

### Available Operations

* [connectProfile](#connectprofile) - Send LinkedIn connection request
* [listInvitations](#listinvitations) - List received LinkedIn connection invitations
* [acceptInvitation](#acceptinvitation) - Accept a LinkedIn connection invitation
* [sendMessage](#sendmessage) - Send LinkedIn message
* [replyToComment](#replytocomment) - Reply to a LinkedIn comment
* [likeComment](#likecomment) - Like a LinkedIn comment
* [publishPost](#publishpost) - Publish or schedule a LinkedIn post
* [createComment](#createcomment) - Comment on a LinkedIn post
* [likePost](#likepost) - Like a LinkedIn post
* [declineInvitation](#declineinvitation) - Decline a connection invitation
* [listSentInvitations](#listsentinvitations) - List sent connection invitations
* [withdrawInvitation](#withdrawinvitation) - Withdraw a sent connection invitation
* [followProfile](#followprofile) - Follow a profile
* [unfollowProfile](#unfollowprofile) - Unfollow a profile
* [editPost](#editpost) - Edit a post
* [editComment](#editcomment) - Edit a comment
* [repostPost](#repostpost) - Repost / share a post
* [unlikePost](#unlikepost) - Unlike a post
* [unlikeComment](#unlikecomment) - Unlike a comment
* [savePost](#savepost) - Save a post
* [unsavePost](#unsavepost) - Unsave a post
* [followCompany](#followcompany) - Follow a company
* [unfollowCompany](#unfollowcompany) - Unfollow a company

## connectProfile

Send a connection request to the specified LinkedIn profile. Rate limited to 80 requests per day per user. 1 credit.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="connectProfile" method="post" path="/connect/linkedin/profile" -->
```typescript
import { Bereach } from "bereach";

const bereach = new Bereach({
  token: "BEREACH_API_KEY",
});

async function run() {
  const result = await bereach.actions.connectProfile({
    profile: "https://www.linkedin.com/in/username",
  });

  console.log(result);
}

run();
```

## acceptInvitation

Accepts a pending connection invitation. Requires the invitationId and sharedSecret obtained from the list invitations endpoint. No daily/weekly cap — only a 5-second minimum interval between calls. Costs 1 credit per request.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="acceptInvitation" method="post" path="/accept/linkedin/invitation" -->
```typescript
import { Bereach } from "bereach";

const bereach = new Bereach({
  token: "BEREACH_API_KEY",
});

async function run() {
  const result = await bereach.actions.acceptInvitation({
    invitationId: "7431723100481851393",
    sharedSecret: "x2mPswjA",
    senderProfileId: "ACoAAB6pI5sBCHuu-3EkvcRl3sgtG7ayAb9FiS0",
    firstName: "Cindy",
    lastName: "Singer",

}

run();
```

## publishPost

### Example Usage

    mode: "instant",
    campaignSlug: "product-launch-2026",
  });
