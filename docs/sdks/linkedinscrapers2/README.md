## Overview

* [collectCommentReplies](#collectcommentreplies) - Scrape replies to a LinkedIn comment

## collectCommentReplies

Returns paginated replies for a specific LinkedIn comment. Use the commentUrn from the comments endpoint response.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="collectLinkedInCommentReplies" method="post" path="/collect/linkedin/comments/replies" -->
```typescript
import { Bereach } from "bereach";

const bereach = new Bereach({
  token: "BEREACH_API_KEY",
});

async function run() {
  const result = await bereach.linkedInScrapers.collectCommentReplies({
    commentUrn: "urn:li:comment:(activity:7654321098765432100,1234567890)",
  });

  console.log(result);
}

run();
```

## getFeed

Get posts from the LinkedIn home feed using GraphQL. Supports algorithmic (RELEVANCE) or chronological (REV_CHRON) sorting. Returns parsed post objects with author info, text, engagement counts. No pagination token support currently. 1 credit.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="getLinkedInFeed" method="post" path="/feed/linkedin" -->
```typescript
import { Bereach } from "bereach";

const bereach = new Bereach({
  token: "BEREACH_API_KEY",
});

async function run() {
  const result = await bereach.linkedInScrapers.getFeed({});

  console.log(result);
}

run();
```
