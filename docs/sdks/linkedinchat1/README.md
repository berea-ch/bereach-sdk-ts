## listInbox

List inbox conversations for the authenticated user. Returns conversations with participants, last message, and read status. Paginate via nextCursor. 0 credits.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="listInbox" method="post" path="/chats/linkedin" -->
```typescript

const bereach = new Bereach({
  token: "BEREACH_API_KEY",
});

async function run() {
  const result = await bereach.linkedinChat.listInbox({});

  console.log(result);
}

run();
```

  token: "BEREACH_API_KEY",
});

    messageUrn: "urn:li:msg_message:(urn:li:fsd_profile:ACoAAXXX,2-MTcw...)",
