# LinkedinActions

## Overview

### Available Operations

* [listInvitations](#listinvitations) - List received LinkedIn connection invitations
* [acceptInvitation](#acceptinvitation) - Accept a LinkedIn connection invitation
* [replyToComment](#replytocomment) - Reply to a LinkedIn comment
* [likePost](#likepost) - Like a LinkedIn post
* [withdrawInvitation](#withdrawinvitation) - Withdraw a sent connection invitation
* [followProfile](#followprofile) - Follow a profile
* [editPost](#editpost) - Edit a post
* [editComment](#editcomment) - Edit a comment
* [repostPost](#repostpost) - Repost / share a post
* [unlikePost](#unlikepost) - Unlike a post
* [unlikeComment](#unlikecomment) - Unlike a comment
* [unsavePost](#unsavepost) - Unsave a post
* [unfollowCompany](#unfollowcompany) - Unfollow a company

## listInvitations

Returns pending connection invitations received by the authenticated LinkedIn account. Each invitation includes the invitationId and sharedSecret needed to accept it via the accept endpoint. Costs 1 credit per request.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="listInvitations" method="post" path="/invitations/linkedin" -->
```typescript
import { Bereach } from "bereach";

const bereach = new Bereach({
  token: "BEREACH_API_KEY",
});

async function run() {
  const result = await bereach.linkedinActions.listInvitations({});

  console.log(result);
}

run();
```

## acceptInvitation

Accepts a pending connection invitation. Requires the invitationId and sharedSecret obtained from the list invitations endpoint. No daily/weekly cap — only a 5-second minimum interval between calls. Costs 1 credit per request.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="acceptInvitation" method="post" path="/accept/linkedin/invitation" -->
```typescript
import { Bereach } from "bereach";

const bereach = new Bereach({
  token: "BEREACH_API_KEY",
});

async function run() {
  const result = await bereach.linkedinActions.acceptInvitation({
    invitationId: "7431723100481851393",
    sharedSecret: "x2mPswjA",
    senderProfileId: "ACoAAB6pI5sBCHuu-3EkvcRl3sgtG7ayAb9FiS0",
    firstName: "Cindy",
    lastName: "Singer",
  });

  console.log(result);
}

run();
```

## replyToComment

### Example Usage

<!-- UsageSnippet language="typescript" operationID="replyToComment" method="post" path="/reply/linkedin/comment" -->

const bereach = new Bereach({
  token: "BEREACH_API_KEY",
});

  const result = await bereach.linkedinActions.replyToComment({
    commentUrn: "urn:li:comment:(activity:7399026485694537728,7399230513867026433)",
    message: "Thank you for your comment!",
    campaignSlug: "lead-magnet-post-12345",

## withdrawInvitation

Withdraw a pending sent connection invitation. Pre-fetches invitee details from last 500 sent invitations, then uses SDUI endpoint to withdraw. Only works for CONNECTION type invitations. Returns bad_request if invitation not found, already withdrawn, or accepted. Returns bad_request with message if invitation expired (410). 1 credit.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="withdrawInvitation" method="post" path="/withdraw/linkedin/invitation" -->
```typescript
import { Bereach } from "bereach";

const bereach = new Bereach({
  token: "BEREACH_API_KEY",
});

async function run() {
  const result = await bereach.linkedinActions.withdrawInvitation({
    invitationUrn: "urn:li:fs_relInvitation:7431723100481851393",
  });

  console.log(result);
}

run();
```

## editPost

Edit the text content of an existing LinkedIn post. Only text can be changed; media, visibility, and other properties are preserved. Requires resolving the shareUrn from the activityUrn (done internally via up to 5 fallback strategies). Only works for posts in the user's recent feed. Returns bad_request if shareUrn cannot be resolved. 1 credit.

### Example Usage

<!-- UsageSnippet language="typescript" operationID="editPost" method="post" path="/edit/linkedin/post" -->
```typescript
import { Bereach } from "bereach";

const bereach = new Bereach({
  token: "BEREACH_API_KEY",
});

async function run() {
  const result = await bereach.linkedinActions.editPost({
    postUrl: "https://www.linkedin.com/feed/update/urn:li:activity:123",
    text: "Updated post text",
  });

  console.log(result);
}

run();
```
